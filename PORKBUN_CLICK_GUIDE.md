# Porkbun: Where to Click for DNS Settings

## Step-by-Step Navigation

### Step 1: Click on roovert.com
On the domain management page, you should see a table with your domain:
- **Click directly on `roovert.com`** (the domain name itself)
- This will take you to the domain details page

### Step 2: Find DNS Settings
Once you click on `roovert.com`, you'll see:
- Domain details page
- Look for tabs or sections like:
  - **"DNS"** tab
  - **"DNS Records"** tab
  - **"DNS Management"** link
  - Or a gear icon (⚙️) with a dropdown menu

### Step 3: Access DNS Records
Click on the DNS/DNS Records section to:
- View current DNS records
- Add new records
- Edit existing records
- Delete records

## What to Look For

After clicking `roovert.com`, you might see:
- A page with tabs at the top (DNS, Email, etc.)
- A sidebar with options (DNS, Email Forwarding, etc.)
- A dropdown menu with "DNS" option
- Or DNS settings directly on the page

## If You Don't See DNS Option

1. Look for a **gear icon (⚙️)** next to the domain
2. Click the gear icon for a dropdown menu
3. Select **"DNS"** or **"DNS Records"** from the menu
